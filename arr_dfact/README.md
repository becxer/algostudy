145 는 재미있는 성질을 가지는 수이다.
145 = 1! + 4! + 5! = 1 + 24 + 120 = 145
n! = n * (n-1) * (n-2) * ... * 1 을 의미하고 0! = 1 로 약속되어 있다.
두 자리수 이상 1000000 이하인 수에서 이런 성질을 가지는 모든 수를 출력하는 것이 문제이다.
