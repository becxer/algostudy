다링 산맥은 많은 목재를 생산한다. 이 목재를 열차에 싣기 전에 , 벌목꾼들은 이를 공터에 쌓아둔다. 
아래 그림은 이를 옆에서 본 그림이다. 

 

이렇게 쌓을 경우 바로 윗단에 있는 통나무의 수는 아래 보다 적어도 하나 이상 적다는 것을 안다. 그리고 각 층의 통나무들은 연결되어 있다. 

윗 그림은 제일 아랫 단의 통나무가 12 개인 경우의 한 예이다. 

제일 아랫 단의 통나무의 수가 주어질 때 , 벌목공들은 이렇게 만들 수 있는 가짓수가 얼마나 되는지를 알고자 한다. 

입력
아랫 단의 통나무의 수 n 이 주어진다.1 <= n <= 2000000000 
출력
수가 크지므로 10^5 로 나눈 나머지를 출력한다. 
입출력 예
입력

1

출력

1

입력

2

출력

2

입력

3

출력

5

입력

5

출력

34
