프로그램 명: election
제한시간: 1 초
5 행 5 열에 H 와 J 로 채워져 있고 , 인접한 7 개 지역에서 J 가 H 보다 많은 가짓 수를 출력하는 것이 문제이다.

인접한 지역이란 가로,세로로 연결되었다는 것을 의미하고 대각선은 인접한 지역으로 인식하지 않는다.

입력

5 행 5 열에 H 과 J 문자가 입력으로 주어진다.
출력

가능한 가짓 수를 출력한다.
입출력 예

입력

HHHHH
JHJHJ
HHHHH
HJHHJ
HHHHH

출력

2
보충 설명

아래의 두가지가 답
.....                .....
JHJHJ                JHJHJ
....H       and      .H...
....J                .J...
.....                .....
