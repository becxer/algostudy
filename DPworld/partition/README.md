프로그램 명: partition

제한시간: 1 초

n 의 분할수란 n 이하의 수들의 합이 n 이 되는 수열의 개수이다.



예를 들어 4 는



1+1+1+1

1+1+2

1+3

2+2

4

의 5 가지 경우가 있다. 즉 , 4 의 분할수는 5 이다.



n 의 분할 수를 p(n) 으로 표기한다면



p(1) = 1

p(2) = 2

p(3) = 3

p(4) = 5

...

이 된다.

입력



n 은 50 이하의 자연수이다.

출력



n 의 분할 수를 출력한다.

입출력 예



입력



5



출력



7
