프로그램 명: bits 

제한시간: 1 초 
 
두 수 n , m 이 입력으로 주어진다. 
?n 은 n bit 를 의미 , 
?m 은 1 의 최대 개수를 의미한다. 

n bit( 2 진수 n 자리) 로 1 의 개수를 m 개 넘지 않는 가짓수가 몇 개인가를 구하는 프로그램을 작성하시오. 

n = 4 ,m = 2 가 주어지면

입력 형식 
두 정수 n(1 <= n <= 31),m(1 <= m <= n) 이 입력으로 주어진다. 
출력 형식 
만들수 있는 총 가지수를 출력한다. 
입출력 예

입력

4 2

출력

11
