프로그램 명: pat(open)
=======================
제한시간: 1 초
두 정수 n , k 를 입력으로 받아 k 개의 1 을 가진 n 자리 이진 패턴을 출력하는 프로그램을 작성 하세요.

입력
두 정수 n , k 가 입력으로 주어진다. (0 < n <= 30 , 0 <= k < 8 , n >= k)

출력
결과를 내림차순으로 출력한다.

입출력 예

입력

2 1

출력

10

01

입력

2 0

출력

00

입력

4 2

출력

1100

1010

1001

0110

0101

0011
